# data-analytics
Coffee Sales Analysis ☕

Project Overview

This project analyzes coffee sales data using Python and Power BI to identify sales trends, customer preferences, and business insights. The data was cleaned and processed using Python, and an interactive dashboard was created in Power BI for visualization.

Objectives

- Analyze coffee sales performance
- Identify top-selling coffee products
- Track sales trends over time
- Understand customer purchasing patterns
- Generate business insights from the data

Tools & Technologies Used

- Python
- Pandas
- NumPy
- Power BI
- Excel Dataset

Workflow

1. Loaded coffee sales dataset from Excel
2. Performed data cleaning and preprocessing using Python
3. Conducted exploratory data analysis (EDA)
4. Created visualizations and insights
5. Built an interactive Power BI dashboard

Dashboard Insights

- Total Revenue: 1,12,245+
- Total Orders: 3,547
- Identified best-selling coffee products
- Sales analyzed by coffee type
- Monthly sales trends
- Day-wise sales analysis
- Time-based sales distribution (Morning, Afternoon, Night)

Files Included

- "analysis.py" → Python script for data analysis
- "Coffee_sales dataset.xlsx" → Dataset file
- "Power BI Report" → Interactive dashboard report
